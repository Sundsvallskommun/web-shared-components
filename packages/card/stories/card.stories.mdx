# Navigationskort

<p>
  Använd kort när du vill göra det enklare för att användaren att få en överblick av innehållet genom att dela upp
  information i olika kategorier.
</p>

### Innehåll

<p>Beroende på vilket behov som ska uppfyllas kan korten innehålla olika typer av information.</p>

#### Korten måste innehålla

- En huvudrubrik
- En kort beskrivande text

#### Annan information som korten kan innehålla

- Bild, illustration eller annat grafiska element
- Knappar
- Externa länkar
- Interna länkar
- Etiketter/taggar
- Datum tid och plats (Nyheter/Evenemang)
