# Att installera biblioteket

<pre>npm install @sk-web-gui/react</pre>

<pre>npm install @sk-web-gui/core</pre>

# Att importera en komponent

Alla komponenter exporteras ifrån @sk-web-gui/react vilket gör att du kan importera önskad komponent såhär:

<pre>{`import { Button } from '@sk-web-gui/react'`}</pre>

och sen använda den:

<pre>{`<Button>Test</Button>`}</pre>

Vilket ger:

<Button>Test</Button>
